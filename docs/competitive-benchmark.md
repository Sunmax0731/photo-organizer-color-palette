# 競合ベンチマーク

        ## 比較対象

        - Google Photos: 採用基準は入力負荷、記録粒度、証跡性、ローカル完結性。
- Adobe Lightroom Mobile: 採用基準は入力負荷、記録粒度、証跡性、ローカル完結性。
- Coolors: 採用基準は入力負荷、記録粒度、証跡性、ローカル完結性。

        ## 採用する基準

        - 入力負荷が低い。
        - 次アクションが明確。
        - ローカルまたは閉域で検証できる。
        - docs と release evidence から状態を追える。

        ## 差別化

        写真選別理由と抽出パレットを制作メモとして残すAndroidツールに絞り、汎用ノートや大型サービスよりも閉域アルファの検証可能性を優先します。
